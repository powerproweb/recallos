How It Works

Set it up in minutes

Five steps. Then your AI handles the rest.

1

Install

pip install recallos
2

Set up your project

recallos init ~/projects/myapp
3

Import your chats

recallos ingest ~/chats/ --mode convos
4

Connect your AI

claude mcp add recallos -- python -m recallos.mcp_gateway
5

Done

Your AI handles the rest. Ask it anything about your past work.

$ pip install recallos
# Set up your project
$ recallos init ~/projects/myapp
✔ Detected 12 nodes from folder structure
✔ Saved recallos.yaml

# Import conversations
$ recallos ingest ~/chats/ --mode convos
✔ Ingested 847 records across 3 domains

$ recallos query "why did we pick Postgres"
→ Found in orion/backend (Nov 15, 2025)
